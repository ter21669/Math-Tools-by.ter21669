# No1. Quadratic equation solution tool - 二次方程式 解の公式算出ツール #
使い方はいたって簡単です。x^2の係数(ただし≠0), xの係数, 定数項を入力すれば、さっと解を算出してくれます。
判別式により区別し、虚数解の場合には解を表示しない方式をとっております。
